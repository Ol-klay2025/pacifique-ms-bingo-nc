Centre de confidentialité

Gérez vos données personnelles et exercez vos droits en matière de protection des données.

Vos droits en matière de protection des données

Conformément au Règlement Général sur la Protection des Données (RGPD), vous disposez des droits suivants concernant vos données personnelles :

  • Droit d'accès : Vous pouvez obtenir une copie de vos données personnelles.
  • Droit de rectification : Vous pouvez faire corriger les données inexactes.
  • Droit à l'effacement : Vous pouvez demander la suppression de vos données dans certaines circonstances.
  • Droit à la limitation du traitement : Vous pouvez demander la restriction du traitement.
  • Droit à la portabilité : Vous pouvez recevoir vos données dans un format structuré.
  • Droit d'opposition : Vous pouvez vous opposer au traitement de vos données.
Données que nous collectons

Données de profil

+

Ces données comprennent votre nom d'utilisateur, adresse e-mail, et préférences de compte.

Ces informations sont utilisées pour vous identifier sur la plateforme et vous contacter.

Données d'authentification et de sécurité

+

Ces données comprennent votre mot de passe (crypté), l'historique des connexions, et informations sur les appareils utilisés.

Ces informations sont utilisées pour sécuriser votre compte et détecter les activités suspectes.

Données de jeu

+

Ces données comprennent votre historique de jeu, cartes achetées, gains et performances.

Ces informations sont utilisées pour gérer les parties, attribuer les gains et améliorer votre expérience de jeu.

Données financières

+

Ces données comprennent votre historique de transactions, méthodes de paiement et solde.

Ces informations sont utilisées pour traiter les paiements, gérer votre solde et respecter les obligations légales.

Données KYC

+

Ces données comprennent les documents d'identité que vous avez fournis pour la vérification de votre compte.

Ces informations sont utilisées pour vérifier votre identité conformément aux exigences réglementaires.

Demander une copie de vos données

Vous pouvez demander une copie de toutes les données personnelles que nous détenons vous concernant. Une fois la demande traitée, vous pourrez télécharger vos données au format JSON.

Le traitement de votre demande peut prendre jusqu'à 30 jours. Vous serez notifié par e-mail lorsque vos données seront prêtes.

Demander la suppression de vos données (droit à l'oubli)

Vous pouvez demander la suppression de vos données personnelles. Cette action est irréversible et entraînera la fermeture de votre compte.

Attention : Certaines données peuvent être conservées même après votre demande de suppression pour respecter nos obligations légales, notamment :

  • Les données relatives aux transactions financières (conservation obligatoire de 5 ans)
  • Les documents d'identité vérifiés dans le cadre du KYC
  • Les données liées aux alertes de sécurité ou de fraude actives
